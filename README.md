#  Globe Hiring – Startup Discovery & Hiring Intelligence Platform

An interactive platform to explore startups across the globe based on **funding stage, hiring activity, and location**, with dynamic filtering and visualization.

---

##  Overview

Globe Hiring is designed to help users discover startups and analyze hiring trends through a **map-based interface**.

The platform focuses on:
-  Global startup discovery
-  Hiring trend insights
-  Advanced filtering & search
-  Interactive visualization

---

- Dynamic UI rendering
